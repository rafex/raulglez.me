# Docs

Documentacion del framework SpecNative Development.

## Contenido

- [`CLI.md`](./CLI.md):
  referencia del CLI de SpecNative, comandos disponibles, campos TOML
  y uso avanzado para proyectos que quieren validacion automatica.

## Separacion importante

Esta carpeta contiene documentacion del framework, no del proyecto.

- Los comandos del proyecto real van en `agents/COMMANDS.md`.
- La arquitectura y convenciones del proyecto van en `agents/`.
- Los procedimientos de ejecucion van en `workflows/`.
